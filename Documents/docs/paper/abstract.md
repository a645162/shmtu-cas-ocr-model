# 摘要

本文针对上海海事大学统一认证平台 (CAS) 算术验证码的自动识别问题，提出了一种基于 **TriSlot Decoder** 的端到端卷积神经网络模型。该验证码格式为「数字-运算符-数字 = 结果」（如 `9-2=7`），传统方法需将图像切割为独立片段后分别送入多个分类器，存在切割误差累积、模型管理复杂等问题。

本文提出的 TriSlot Decoder 架构采用共享 CNN backbone 提取空间特征图，通过 3 个可学习的槽位查询 (Slot Query) 基于多头注意力机制对特征图进行位置感知的聚合，单次前向传播即可同时输出左数字、运算符、右数字三个分类结果。同时，本文设计了包含槽位顺序约束、重叠惩罚、右边界约束和注意力方差约束的结构化损失函数，引导模型自发学习验证码字符的空间排列规律。

在数据采集方面，本文构建了基于 CAS 验证码接口的自动化采集管线，采用「先 OCR 后验证」的策略实现半自动标注，并公开了包含数千条标注数据的验证码数据集。数据增强方面，针对验证码图像的形态特征，设计了平移、仿射变换、形态学操作、稀疏噪点和二值化参数扰动等策略。

实验结果表明，以 MobileNetV3-Small 为 backbone 的 TriSlot Decoder 模型在验证集上达到 **98%+ 的整体表达式准确率**，其中运算符分类准确率超过 99.7%。模型可导出为 ONNX 和 NCNN 格式，在 CPU 上实现约 5ms/图的推理速度，满足实时部署需求。与 V1 三模型分离方案相比，V2 将推理效率提升约 3 倍，部署复杂度显著降低。

**关键词**：验证码识别；槽位注意力；端到端学习；卷积神经网络；轻量化部署
